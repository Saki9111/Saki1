# Taxi City Zagreb - Changelog

Sve promjene na projektu su dokumentirane u ovoj datoteci.

## [1.0.0] - 2026-07-23

### Dodano (Added)
- Inicijalni Android projekt
- Google Maps integracija
- Firebase konfiguracija (Auth, Firestore)
- MVVM arhitektura
- Jetpack Compose UI framework
- Modeli podataka (User, Ride, PricingInfo, TaxiLocation)
- Utility funkcije (LocationUtils, Constants)
- Repository slojevima (RideRepository, PricingRepository, UserRepository)
- Tema i boje (TaxiCityZagrebTheme)
