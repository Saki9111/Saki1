# Taxi City Zagreb - Android Aplikacija

Android aplikacija za uslugu prijevoza - taxi rezervacije s izračunom cijene, Google Maps navigacijom i rutama.

## Značajke
- ✅ Brza rezervacija taksija
- ✅ Izračun cijene u realnom vremenu
- ✅ Google Maps navigacija
- ✅ Praćenje vozila (real-time)
- ✅ povijest rezervacija
- ✅ Profil korisnika

## Tehnologija
- **Jezik:** Kotlin
- **Arhitektura:** MVVM
- **UI Framework:** Jetpack Compose / Material Design 3
- **Backend:** Firebase (Firestore, Authentication)
- **Mape:** Google Maps SDK
- **HTTP Client:** Retrofit
- **Lokalna baza:** Room
